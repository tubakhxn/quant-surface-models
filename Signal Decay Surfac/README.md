# Signal Decay Surface

## dev/creator = tubakhxn

This project demonstrates a professional quant visualization of signal decay, regression modeling, and advanced plotting in Python.

---

## How to Fork

1. On GitHub, click the "Fork" button at the top right of the repository page.
2. Choose your GitHub account as the destination.
3. Clone your forked repository to your local machine:
   
   ```sh
   git clone https://github.com/your-username/Signal-Decay-Surface.git
   ```

---

## How to Run

1. Ensure you have Python 3.7 or newer installed.
2. Open a terminal in the project directory.
3. Run the script:
   
   ```sh
   python main.py
   ```
   The script will auto-install all required dependencies and display all plots.
