## How to Fork

1. Click the "Fork" button on the top right of the repository page on GitHub.
2. Choose your GitHub account as the destination for the fork.
3. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/your-username/Regime-Transition-Curve-System.git
   ```
4. Navigate into the project directory:
   ```bash
   cd Regime-Transition-Curve-System
   ```

## How to Run

1. Make sure you have Python 3.7 or higher installed.
2. Run the main script (it will auto-install dependencies):
   ```bash
   python regime_transition_curve_system.py
   ```
3. The script will display all plots and print a completion message.
