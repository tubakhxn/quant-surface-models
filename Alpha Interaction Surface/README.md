## How to Fork

1. Click the "Fork" button at the top right of the repository page on GitHub.
2. Choose your GitHub account as the destination.
3. Clone your fork to your local machine:
   ```sh
   git clone https://github.com/YOUR_USERNAME/Alpha-Interaction-Surface.git
   ```
4. Navigate into the project directory:
   ```sh
   cd Alpha-Interaction-Surface
   ```

---

## How to Run

1. Make sure you have Python 3.8 or newer installed. ([Python Download](https://www.python.org/downloads/))
2. Open a terminal in the project directory.
3. Run the main script:
   ```sh
   python main.py
   ```
   The script will automatically install all required dependencies and display all plots.
